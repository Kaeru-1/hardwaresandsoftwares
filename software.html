The system software is a collection of programs designed to operate, control, andextend the processing capabilities of the computer itself.

System software serves as the interface between the hardware and the end-users.

Application software products are designed to satisfy a particular need of a particularenvironment.
Application software may consist of a singleprogram, or it may also consist of a collection of programs, often called a software package

All software applications prepared in the computer lab can come under the category of Application software. through which all components and external peripherals connect. Motherboards can be found in virtually all computers, especially desktop and laptop PCs.
